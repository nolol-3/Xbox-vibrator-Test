# Xbox-vibrator-Test

To make the script work, you only need to download One Python lib.

Open your Command Prompt (CMD) or PowerShell and paste this exact command:

pip install soundcard numpy

And last version of python,

What do these do?

soundcard: This is the magic library that secretly taps into your USB audio device so it can hear the sound playing.

numpy: This does the fast math to calculate how loud the audio is so the script knows how hard to vibrate the controller.

What you DO NOT need to download:

No Xbox drivers: Windows already has the Xbox 360 controller drivers built-in. Just plug it in.

Quick Checklist:
Controller plugged in via USB or official Xbox PC Wireless Adapter.
Run the pip install command above.
Run the Python script.
Type devices, type the number for your USB device, and type vibrate_when_loud.
